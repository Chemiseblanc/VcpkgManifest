# VcpkgManifest.psm1
A powershell module for manipulating vcpkg manifest files

## Commands
The details for using each command can be retrieved using Get-Help
- New-VcpkgManifest
- Update-VcpkgManifest
- Add-VcpkgDependency
- Remove-VcpkgDependency
- Update-VcpkgDependency
- Read-VcpkgManifest
- Write-VcpkgManifest
